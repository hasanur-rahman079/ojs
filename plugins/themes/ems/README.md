# EMS Theme for OJS 3.5

A modern, professional theme for Open Journal Systems (OJS) 3.5 based on the complete ems.pub Figma design.

## 🎨 Design Source

**Figma Design**: https://www.figma.com/design/v7FGciVqOvd7Cl45tvS2da/ems.pub?node-id=0-1

## ✨ Why This Theme?

### 🎯 **Professional Design**
- Pixel-perfect implementation of the ems.pub Figma design
- Modern, clean interface that builds trust with authors and readers
- Consistent branding across all pages

### 📱 **Fully Responsive**
- Beautiful on desktop, tablet, and mobile devices
- Touch-friendly interface for mobile users
- Adaptive layouts for all screen sizes

### ⚡ **User-Friendly**
- Intuitive navigation and forms
- Clear visual hierarchy
- Smooth interactions and animations

### 🔧 **Easy to Use**
- Context-aware header (adapts for site vs journal pages)

### 🌟 **Key Features**
- **Landing Page**: Complete marketing page with hero, features, pricing
- **Registration**: Modern form with ORCID integration
- **Login**: Centered card design with password toggle
- **Journal Pages**: Professional journal home and submission pages
- **Clean URLs**: Proper URL structure for better SEO

## 📦 Installation

1. Copy the `ems` folder to `plugins/themes/` in your OJS installation
2. Log into OJS admin panel
3. Go to **Settings → Website → Appearance**
4. Select **EMS Theme** and click **Enable**

## 🎨 Design Highlights

- **Colors**: Professional green/teal color scheme (#0abf96, #019863)
- **Typography**: Modern fonts (Manrope, Work Sans) from Google Fonts
- **Layout**: Clean, spacious designs with proper spacing
- **Components**: Reusable buttons, cards, and form elements

The theme supports OJS admin customization:

- Logo and branding
